# CE Structure to C++ Converter

## 功能特性
- ✅ 可视化编辑 CE 结构体
- ✅ 自动映射 C# 泛型类型
- ✅ 修改回写到 XML
- ✅ 生成标准 C++ 头文件
- ✅ 循环引用检测

## 使用方法
1. 打开 .csx 文件
2. 双击字段改名
3. 右侧面板调整类型
4. 点击 Generate 生成 C++ 代码

## 输出示例
生成的 C++ 代码自动包含 `CSharpTypes.h`
